v4.0 Architecture Overview

The 3-Layer Score System

Rather than a simple blended score, v4.0 isolates data points based on their cyclical timing. Predictive Score (50%) uses Leading indicators to project where the cycle will be 3-6 months from now. Diagnostic Score (30%) uses Coincident data to show current conditions. Confirmation Score (20%) waits for Lagging data to validate regime transitions.

Demand Decoupling (AI: 70%, Consumer: 30%)

The Demand dimension is heavily weighted toward AI Infrastructure (Hyperscaler CapEx, HBM Premium) because it drives over 90% of memory bit growth. Traditional consumer data acts as a baseline.

Regime Engine Hysteresis

To stop the 'expansion/late cycle' rapid toggling seen in v3.1, Regime Engine v4.0 uses hysteresis. Entering Expansion requires a higher score threshold (67) than exiting it (63).

New Macro Variables

High Yield Spreads: Directly tracks corporate credit risk. A widening spread predicts tech derating.

Sahm Rule: Real-time recession indicator historically proving to be an unbreakable leading warning of semiconductor demand collapse.
